# Sfera grpcChat App

## Set Up

add

`127.0.0.1 grpc.local`

to

`etc/hosts`

## Installation

Run

`git clone https://github.com/sbbsworks/grps-chat.git`

`cd grps-chat`

`docker compose build`

`docker compose up -d`

`npm install`

`npm run build`

## Open

[http://grpc.local:5050](http://grpc.local:5050)
